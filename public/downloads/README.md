# Jay's Phone Repair Mobile App

## Download Instructions

### For Android:
The Android APK is currently being prepared. 

To build the APK:
1. Open terminal in the mobile-app folder
2. Run: `npm install -g eas-cli`
3. Run: `eas login` (create free Expo account)
4. Run: `eas build --platform android --profile preview`
5. Wait 10-15 minutes for the build
6. Download the APK and place it here

### For iOS:
Use Safari on your iPhone:
1. Visit this website
2. Tap the Share button
3. Tap "Add to Home Screen"
4. The app will install like a native app!
